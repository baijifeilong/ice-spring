> 最纯的播放器给最纯的你

## 简介

冰泉是一个支持灵活布局的音乐播放器

## 下载

- [Github](https://github.com/baijifeilong/ice-spring/releases)

## 使用

- 拖动单个或多个文件或文件夹到应用窗体
- 如果存在同名`.lrc`歌词文件，会同步显示歌词
